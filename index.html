This HTML Cheatsheet is For You!
  1. HTML Boilerplate: This is the basic html boilerplate code 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
  2. List Item Element Ordered list and Unordered List 
    <ol>
<li>"Head east on Prince St"</li>
<li>"Turn left on Elizabeth"</li>
</ol>

<ul>
<li>"Cookies"</li>
<li>"Milk"</li>
</ul>
  3. Em Tag: Emphasis Element 
    <p>"This <em>word</em> will be emphasized in italics."</p>
  4. Video Tag This is the video element 
    <video src="test-video.mp4" controls>
Video not supported
</video>
  5. Img Tag This is the image element 
    <img src="image.png">
  6. Div tag: The element is used as a container that divides an HTML document into sections 
    <div>
<h1>"A section of grouped elements"</h1>
<p>"Here’s some text for the section"</p>
</div>
<div>
<h1>"Second section of grouped elements"</h1>
<p>"Here’s some text"</p>
</div>
  7. H1-H6: These are the Heading elements 
    <h1>"Breaking News"</h1>
<h2>"This is the 1st subheading"</h2>
<h3>"This is the 2nd subheading"</h3>
<h4>"This is the 3rd subheading"</h4>
<h5>"This is the 4th subheading"</h5>
<h6>"This is the 5th subheading"</h6>
  8. Span Tag: The element is an inline container for text and can be used to group text for styling purposes 
    <p><span>"This text</span> may be styled differently than the surrounding text."</p>
  9. Strong element: The element highlights important, serious, or urgent text and BOLD the text 
    <p>"This is <strong>important</strong> text!"</p>
  10. Anchor Tag: The anchor element is used to create hyperlinks in an HTML document 
    <!-- Creating text links -->
<a href="http://www.codecademy.com">Visit this site</a>
                    
<!-- Creating image links -->
<a href="http://www.codecademy.com">
<img src="logo.jpg">Click this image
</a>
  11. Table Tag: The element has content that is used to represent a two-dimensional table made of rows and columns 
    <table>
<!-- "rows and columns will go here" -->
</table>
  12. Style Tag: This is used to define style information (CSS) for a document 
    <style>
#first {
font-family: Castellar;
background-color: green;
color: white;
}                  
.second {
text-align: center;
background-color: white;
font-size: 30px;
color: red;
}
</style>
  13. Sub and Sup Tag: This is used when to write the text as subscript or superscript 
    <div>MANSI<sub>subscript text</sub></div>
<!-- superscript -->
<div>MANSI<sup>Superscript text</sup></div>
  14. Form Tag: It represents a document section containing interactive controls for submitting information 
    <form action="/action_page.php" method="get">
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname"><br><br>
<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname"><br><br>
<input type="submit" value="Submit">
</form>
  15. Textarea Tag: It defines a multi-line text input controls 
    <textarea id="w3review" name="w3review" rows="4" cols="50">
"This is a basic webpage for HTML cheatsheet codes."
"This will help you work faster and easily copy paste code snippets to you HTML file."
</textarea>
  16. Option Tag: It gives a dropdown menu to select options 
    <label for="cars">Choose a car:</label>
<select id="cars">
<option value="volvo">Volvo</option>
<option value="saab">Saab</option>
<option value="opel">Opel</option>
<option value="audi">Audi</option>
</select>
  17. Blockquote Tag: It specifies a section that is quoted from another source 
    <blockquote cite="http://www.worldwildlife.org/who/index.html">
"For 50 years, WWF has been protecting the future of nature." 
"The world's leading conservation organization, WWF works"
"in 100 countries and is supported by 1.2 million members"
"in the United States and close to 5 million globally."
</blockquote>
  18. Del Tag: It defines text that has been deleted from a document 
    <p>My favorite color is <del>blue</del> <ins>red</ins>!</p>
  19. Footer Tag: It defines a footer for a document or section 
    <footer>
<p>Author: Hege Refsnes</p>
<p><a href="mailto:hege@example.com">hege@example.com</a></p>
</footer>
  20. Fieldset Tag: It is used to group related elements in a form and draws a box around them 
    <form action="/action_page.php">
<fieldset>
<legend>Personalia:</legend>
<label for="fname">First name:</label>
<input type="text" id="fname" name="fname"><br><br>
<label for="lname">Last name:</label>
<input type="text" id="lname" name="lname"><br><br>
<label for="email">Email:</label>
<input type="email" id="email" name="email"><br><br>
<label for="birthday">Birthday:</label>
<input type="date" id="birthday" name="birthday"><br><br>
<input type="submit" value="Submit">
</fieldset>
</form>
